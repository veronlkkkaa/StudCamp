# StudCamp

Мобильное приложение для общения в учебных группах в реальном времени. Любой участник может создать комнату и стать её хостом — встроенный сервер поднимается прямо на устройстве. Остальные подключаются по локальной сети.

## Что умеет

- Создавать и присоединяться к чат-комнатам по локальной сети
- Общаться в реальном времени через WebSocket
- Обмениваться файлами внутри комнаты
- Регистрироваться и входить под своим аккаунтом
- Редактировать профиль

## Как это работает

Устройство-хост запускает Ktor-сервер и объявляет комнату через NSD (Network Service Discovery). Остальные участники находят комнату автоматически и подключаются по WebSocket. Сообщения и файлы идут через этот же канал.

## Стек

- **UI** — Jetpack Compose
- **Сеть** — Ktor Client + Ktor Server (встроенный)
- **Навигация** — Navigation Compose
- **Сериализация** — kotlinx.serialization

## Структура

```
app/          # точка входа, встроенный сервер (Ktor), файловое хранилище
core/         # модели, сетевой клиент, репозитории, навигация, общие UI-компоненты
feature/
  auth/       # регистрация и вход
  chat/       # список комнат, чат, информация о комнате
  room/       # создание и подключение к комнате
  profile/    # просмотр и редактирование профиля
```

## Запуск

Откройте проект в Android Studio и запустите на устройстве или эмуляторе с Android 7.0+ (API 24).
